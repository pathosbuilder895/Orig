Submit Baseline Essays
Help Original learn each student's authentic voice
What is a baseline? A baseline is an essay you know is definitely from the student. Original uses 3+ baselines to build a writing fingerprint for each student.
Three types:
  • Proctored: Written in class under supervision (strongest baseline)
  • Verified: Confirmed authentic by student or parent (medium strength)
  • Unverified: Assumed authentic from earlier assignments (weakest)
Proctored (Strongest)
Written in class with supervision. Most reliable for building fingerprint.
Verified (Medium)
Student confirmed as their authentic work via email or in-person.
Unverified (Weak)
Previous assignments assumed to be the student's work. Use as backup.

You can add more baselines later through the dashboard

Reading Your Results
A quick tour of the Results page before you begin

✓ Setup Complete

Original is ready. Here's how to read the results page when a submission is flagged.

🟢🟡🟠🔴
Authorship Confidence Badge
The large colored badge at the top of every review. Green = Authentic, Yellow = Monitor, Orange = Discuss, Red = Review Warranted. Your at-a-glance verdict — but read the plain-English summary first.
📝
Plain-English Summary
Next to the badge, Original describes what it found in plain language — which specific features deviated from the student's baseline. Always read this before looking at numbers.
📊
Feature Interference Map
A bar chart showing which features are suspicious (red → right) vs. authentic (green ← left). Longer red bars = bigger anomalies. Each bar links to a plain-English explanation.
📈
Writing Trajectory
A sparkline showing the student's last 6 submissions. Gradual drift = natural growth. A sudden jump at the end is the key pattern to investigate.
⚖️
Your Judgement — You Always Decide
Original never accuses. Use the 5-step corroboration checklist, then choose: Clear, Monitor, Discuss, or Escalate. Your decision and notes are saved permanently.
Need a reminder later? The Help page explains every element of the results page in detail.
